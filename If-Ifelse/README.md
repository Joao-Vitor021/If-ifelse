# Sistema de Aprovação — Java

Programa simples em Java que simula um **sistema de aprovação escolar**,
utilizando a nota informada pelo usuário para determinar o resultado final.

## Como funciona
- Solicita uma nota de **0 a 10**
- Valida se a nota é válida
- Exibe o resultado:
  - **Aprovado** (nota ≥ 7)
  - **Recuperação** (nota ≥ 5)
  - **Reprovado** (nota < 5)

## Conceitos praticados
- Estruturas condicionais (`if`, `else if`, `else`)
- Validação de dados de entrada
- Uso da classe `Scanner`
- Controle de fluxo do programa

## Tecnologias
- Java  
- Aplicação Console

## Objetivo
Projeto desenvolvido para reforçar a **lógica de programação em Java** e o entendimento do **fluxo do código**.
